# Emotional AI - A Replika AI Clone

Emotional AI is a conversational AI chatbot inspired by Replika AI, designed to engage in personalized conversations with users, recognize emotions, and build relationships through natural language interactions.

## Features

- **Personalized Conversations**: Adapts to user's personality and preferences over time
- **Emotion Recognition**: Identifies emotions in user messages and responds appropriately
- **Memory System**: Remembers past conversations and important user details
- **User Profiles**: Stores user preferences and conversation history
- **Natural Language Processing**: Understands and generates human-like responses

## Technology Stack

- **Backend**: Node.js with Express
- **Database**: SQLite for storing conversations and user data
- **NLP**: Natural language processing for understanding user intent and emotions
- **Frontend**: React for the user interface

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm start
   ```
4. Open your browser and navigate to `http://localhost:3000`

## Project Structure

- `/src` - Source code
  - `/api` - Backend API endpoints
  - `/components` - React components
  - `/models` - Data models
  - `/services` - Business logic
  - `/utils` - Utility functions
- `/public` - Static assets
- `/database` - Database schema and migrations
